# Japanese-urushi-no-ma

漆ノ間（Urushi no Ma）は、架空の日本漆器工芸室を題材にした、Next.js + React + TypeScript 製のレスポンシブWebサイトです。

## Features

- GSAP ScrollTriggerによる漆の刷毛跡ヒーローアニメーション
- 漆器の技法、作品紹介、お手入れ、工房の物語、問い合わせ、法務ページ
- 日本語コンテンツと架空サイト・素材安全メモの明示
- オリジナル生成画像を使った工芸ビジュアル
- 軽量なCSS/SVG表現、遅延読み込み画像、レスポンシブ対応

## Commands

```bash
npm install
npm run dev
npm run build
```

## Disclaimer

このサイトは架空の伝統工芸スタジオ紹介サイトです。実在する工房、職人、販売店、注文・決済機能を示すものではありません。
